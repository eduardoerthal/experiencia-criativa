## Descrição

Este projeto foi desenvolvido como parte de uma experiência criativa na PUCPR, com foco na criação de um site de vendas de produtos cosméticos femininos. O objetivo é oferecer uma plataforma online intuitiva e funcional, onde as consumidoras possam encontrar e adquirir produtos de beleza com facilidade e confiança.

## Funcionalidades

- **Catálogo de produtos**: Exibição de diversos produtos cosméticos com descrições detalhadas.
- **Carrinho de compras**: Funcionalidade para adicionar e remover itens do carrinho.
- **Checkout simplificado**: Processo de finalização de compra fácil e rápido.
- **Busca de produtos**: Sistema de busca eficiente para encontrar produtos por categoria ou nome.
- **Design responsivo**: O site foi projetado para ser acessado de qualquer dispositivo, garantindo uma experiência otimizada para o usuário.
